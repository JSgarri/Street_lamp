## Street Lamp NTP
Device to control a street lamp conected to a NTP server to obtain the hour.

## Required libraries:
	Time-Master (https://github.com/PaulStoffregen/Time) 
	Time by Michael Margolls (included on Arduino Manage Libraries)
## Pin-Out
* Relay     -->  2
* Led Red   -->  3
* Led Green -->  5
* ds18b20   -->  6
